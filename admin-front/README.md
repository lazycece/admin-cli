# reusable-vue-admin
Reusable vue admin

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9090
npm run dev

# build for test environment
npm run build:sit

# build for production environment
npm run build:prod

# build for production and view the bundle analyzer report
npm run build:prod --report
```
